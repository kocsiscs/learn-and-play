# 🌟 Learn and Play

A collection of interactive learning games for kids, built with vanilla HTML, CSS, and JavaScript.

## Games

### ✨ Helyesírás Varázs *(Spelling Magic)*
Practice Hungarian spelling in a fun and engaging way. Work through letters, words, and sentences across multiple difficulty levels.

### 🔢 Matek Varázs *(Math Magic)*
Sharpen your arithmetic skills with addition, subtraction, multiplication, and division — all wrapped in a playful experience.

## Play Online

👉 **[Open the game hub](https://kocsiscs.github.io/learn-and-play/)**

## Tech Stack

- Pure HTML, CSS, JavaScript — no dependencies
- Responsive design, works on desktop and mobile
- Dark theme with animated UI
